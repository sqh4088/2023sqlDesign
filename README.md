# 2023sqlDesign
山东大学（威海）数据库课程设计项目，某学校的题库管理系统
## 简介
笔者学识浅薄，不会先进技术，仅仅利用课内所学的Java Swing完成了大二下的数据库课程设计。  
倘若有需要用以应付课程的，本项目可以说完全够用，当然，笔者为此也付出了不少心血，  
所以，谢绝白嫖，杜绝白嫖。  
联系笔者，有偿获取整个项目完整代码，以及笔者亲自所撰写的前端界面设计文档以及后台数据库设计文档。  
+Q 3317770737 带着可接受的价格来。
